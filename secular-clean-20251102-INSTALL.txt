SECULAR - INSTALLATION INSTRUCTIONS
====================================

Package: secular-clean-20251102.tar.gz (5.2MB)

PREREQUISITES
-------------
1. Radicle CLI (rad command)
   - Install: https://radicle.xyz/install
   - Or: brew install radicle

2. Rust toolchain (for CLI)
   - Install: curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

3. Node.js 18+ (for GUI)
   - Install: https://nodejs.org/
   - Or: brew install node


INSTALLATION STEPS
------------------

1. Extract the package:
   tar -xzf secular-clean-20251102.tar.gz
   cd Secular

2. Install the CLI (Optional):
   cd secular-cli
   cargo build --release
   cargo install --path .
   cd ..

   # Test it works:
   secular --version

3. Install the GUI (Optional):
   cd secular-gui
   npm install
   cd ..

4. Start your Radicle node:
   rad node start

   # Check node status:
   rad node status


RUNNING THE APPLICATION
------------------------

Option A: Use the CLI
   secular --help
   secular friend list
   secular repos list

Option B: Use the GUI
   cd secular-gui
   npm run dev
   # Opens at http://localhost:5173

   # Or start the full stack:
   node server.js &
   npm run dev

TROUBLESHOOTING
---------------
- If "rad" command not found: Install Radicle CLI first
- If "cargo" not found: Install Rust toolchain
- If "npm" not found: Install Node.js
- If node won't start: Check if another Radicle node is already running
